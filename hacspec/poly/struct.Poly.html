pub struct Poly<T: TRestrictions<T>> { /* fields omitted */ }

The poly struct. Note that while this is a polynomial over ℤn[x]/mℤ[x] it is not necessarily reduced by mℤ[x], i.e. poly might be larger. Call reduce to make sure poly is in ℤn/mℤ.
